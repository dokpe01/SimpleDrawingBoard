# 3. Key Features (주요 기능)
- 캔버스에 다양한 색상의 브러쉬를 마우스로 하여금 그림을 손 쉽게 그릴 수 있다.
- 적당한 두께의 지우개를 이용하여 그렸던 그림을 간단히 지울 수 있다.
- 그린 그림을 sdcard 에 저장하고 사용자가 직접 확인할 수 있다.
- 새로 만들기 버튼을 클릭하면 새 캔버스를 불러와 다시 그림을 그릴 수 있다.
